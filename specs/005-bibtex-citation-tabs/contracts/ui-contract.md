# UI Contract: BibTeX Citation Section

## Inputs

- Two BibTeX entries with labels "Paper" and "Dataset".
- Optional placeholder text for missing entry content.

## Outputs

- Visible BibTeX code block for the selected entry.
- Copy action that places the selected entry text on the clipboard.
- Inline success feedback near the copy button.

## Interaction Rules

- Tabs switch the visible entry.
- Full keyboard navigation for tabs and copy action.
